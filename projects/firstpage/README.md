### Tvorba webu s real-time náhledem
1. Otevři VS Code.
2. Nainstaluj si rozšíření Live Server.
![alt Live Server](Live-Server.png)
3. Vpravo dole ve VS Code klikni na Go Live.
4. Vytvářej web a sleduj vývoj v reálném čase.
